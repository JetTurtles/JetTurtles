Windows安装Linux环境

使用 wls 在 Windows 安装 Ubuntu（Microsoft官网）

  1. 在 Windows 功能中选中以下两个选项

  2. 根据提示重启电脑

  3. 打开命令行查看wsl版本并更新

  4. 打开Microsoft Store，搜索 Ubuntu 20.04.6 LTS并点获取
